# Split-A-Bill (Splitable)
An app to easily split bills amongst the group to ensure each one pays for what they had.

## Features
- Propotional Split logic implemented, i.e, users can input their portion size consumed for a particular item and split is calculated accordingly. 
- Generates a PDF to share with people with whom the bill is split.
